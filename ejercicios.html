Documentos del Proyecto

Aquí puedes encontrar la documentación detallada de los ejercicios y recursos educativos del proyecto.

Recursos Educativos y Ejercicios

  • Recurso Educativo 12 (PDF): Guía paso a paso para simular un cubo OLAP usando Pentaho Data Integration (versión 8) y un archivo CSV. El ejercicio cubre las operaciones OLAP clave: Drill Down (profundizar de trimestre a mes), Roll Up (resumir de mes a trimestre), Slice (filtrar por un solo mes como Mayo) y Dice (filtrar por múltiples condiciones como mes, región y producto). Incluye instrucciones detalladas para configurar Pentaho Spoon y visualizar los resultados, así como una interpretación de cada operación.
  • Recurso Educativo 16 (PDF): Guía práctica para crear un dashboard interactivo utilizando Chart.js, HTML y datos en formato JSON. El documento detalla cómo procesar datos de ventas y productos con Pentaho para generar un archivo JSON (`analisis_ventas.json`), y luego cómo visualizar los ingresos totales por categoría de producto en un gráfico de barras. Incluye el código HTML y JavaScript necesario para el dashboard, así como instrucciones para ejecutarlo usando un servidor local de Python.
  • Recurso Educativo 19 (PDF): Guía paso a paso para construir un flujo ETL en Pentaho Spoon. El objetivo es leer un fact (ventas) y tres dimensiones (producto, cliente, tienda) desde archivos CSV, realizar uniones (joins) entre ellos sin necesidad de una base de datos, y generar datos agregados por año, mes y categoría de producto. El documento detalla el uso de pasos como Text file input, Sort rows, Merge Join, Modified Java Script Value para cálculos (sales_amount) y extracción de partes de la fecha, Group By para agregaciones, y Text file output para generar archivos CSV de salida. Incluye configuraciones clave y tips técnicos para evitar errores comunes.
  • Recurso Educativo 20 (PDF): Guía paso a paso para poblar una base de datos HSQLDB local con datos de archivos CSV utilizando Pentaho Data Integration (PDI). El documento detalla la creación de una conexión a HSQLDB, la ejecución de scripts DDL para crear tablas de dimensiones (product_dim, store_dim, date_dim) y una tabla de hechos (sales_fact), y la carga de datos desde CSV a estas tablas. Además, incluye una serie de consultas SQL de ejemplo (básicas, intermedias con joins y agregaciones, y avanzadas de análisis tipo OLAP) para practicar la extracción y análisis de datos desde la base de datos poblada.
  • Recurso Educativo 26 (PDF): Guía completa para realizar operaciones OLAP (Drill-down, Roll-up, Slice & Dice, Pivot) utilizando Pentaho Data Integration 7.1. El documento detalla la creación de archivos CSV de ejemplo (producto, tienda, fecha, venta), la configuración de una base de datos HSQLDB local con scripts SQL para crear tablas, y la carga de datos desde CSV. Para cada operación OLAP, se proporcionan consultas SQL de ejemplo y explicaciones sobre cómo implementarlas y analizarlas, utilizando un caso de estudio de ventas de una cadena de tiendas.
  • Recurso Educativo 30 (PDF): Guía para simular un cubo OLAP utilizando archivos CSV y Pentaho Data Integration (PDI). El documento describe cómo diseñar un flujo ETL en Spoon para leer datos de ventas, clientes y productos desde archivos CSV, unirlos mediante pasos 'Join Rows', y luego agregar dimensiones y medidas utilizando un paso 'Group By' para simular consultas OLAP. Finalmente, se exportan los resultados agregados a un archivo CSV, que actúa como una 'vista OLAP' para análisis posteriores.
