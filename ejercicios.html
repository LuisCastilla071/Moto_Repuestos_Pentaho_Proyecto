Documentos del Proyecto

Aquí puedes encontrar la documentación detallada de los ejercicios y recursos educativos del proyecto.

Recursos Educativos y Ejercicios

Recurso Educativo 12: Simulación de Cubo OLAP

Guía paso a paso para simular un cubo OLAP usando Pentaho Data Integration (versión 8) y un archivo CSV. El ejercicio cubre las operaciones OLAP clave: Drill Down (profundizar de trimestre a mes), Roll Up (resumir de mes a trimestre), Slice (filtrar por un solo mes como Mayo) y Dice (filtrar por múltiples condiciones como mes, región y producto).

Abrir PDF

Recurso Educativo 16: Dashboard Interactivo con Chart.js

Guía práctica para crear un dashboard interactivo utilizando Chart.js y HTML. El documento detalla cómo procesar datos de ventas y productos con Pentaho para generar un archivo JSON (`analisis_ventas.json`), y luego cómo visualizar los ingresos totales por categoría de producto en un gráfico de barras.

Abrir PDF

Recurso Educativo 19: Flujo ETL desde CSV

Guía paso a paso para construir un flujo ETL en Pentaho Spoon. El objetivo es leer un fact (ventas) y tres dimensiones (producto, cliente, tienda) desde archivos CSV, realizar uniones (joins) entre ellos sin necesidad de una base de datos, y generar datos agregados por año, mes y categoría de producto.

Abrir PDF

Recurso Educativo 20: ETL a Base de Datos HSQLDB

Guía paso a paso para poblar una base de datos HSQLDB local con datos de archivos CSV utilizando Pentaho Data Integration (PDI). El documento detalla la creación de una conexión a HSQLDB, la ejecución de scripts DDL para crear tablas y la carga de datos desde CSV a estas tablas.

Abrir PDF

Recurso Educativo 26: Operaciones OLAP con PDI 7.1

Guía completa para realizar operaciones OLAP (Drill-down, Roll-up, Slice & Dice, Pivot) utilizando Pentaho Data Integration 7.1. El documento detalla la creación de archivos CSV de ejemplo, la configuración de una base de datos HSQLDB local y la carga de datos desde CSV.

Abrir PDF

Recurso Educativo 30: Simulación de Cubo OLAP con Múltiples CSV

Guía para simular un cubo OLAP utilizando archivos CSV y Pentaho Data Integration (PDI). El documento describe cómo diseñar un flujo ETL en Spoon para leer datos de ventas, clientes y productos desde archivos CSV, unirlos mediante pasos 'Join Rows', y luego agregar dimensiones y medidas.

Abrir PDF
