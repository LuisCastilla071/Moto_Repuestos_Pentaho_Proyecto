Visualización Gráfica de Datos (P5)

Gráficos interactivos generados con Chart.js para representar los resultados del análisis de datos.

Ventas por Mes

Explicación Técnica:

Este gráfico de barras muestra el total de ventas para cada mes. Los datos se obtienen de un archivo JSON y se representan utilizando la librería Chart.js.

Explicación Técnica:

Este gráfico de barras muestra el total de ventas para cada categoría de producto. Los datos se obtienen de un archivo JSON y se representan utilizando la librería Chart.js.
